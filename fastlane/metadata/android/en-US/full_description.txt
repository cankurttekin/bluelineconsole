andmenu is a lightweight, keyboard-centric launcher designed for blazing fast speed and efficiency. 
Inspired by tools like `dmenu` and `rofi`, andmenu lets you launch apps, search the web, calculate formulas, and run quick commands.

Blazing-Fast App Search: Type 2 or 3 characters to find and launch any app instantly.
Fuzzy Search: Don't worry about exact spelling—find apps quickly even with typos or partial names.
Usage Frequency Sorting: Your most-used applications automatically rise to the top for faster access.
App Shortcuts & Management: Long-press any app to access system app info, uninstall, hide apps from the list, or rename them.
Custom Themes: Style your launcher to fit your aesthetic with themes like dmenu, Gruvbox, Rosé Pine and more.


Commands:

- help
- config
- date
- ping HOST
- ping6 HOST
- all_apps
- hidden_apps
- sys [wifi, bluetooth, mobile_data, location, airplane_mode]
- bing QUERY    (Bing search)
- d QUERY       (DuckDuckGo)
- google QUERY  (Google)
- ecosia QUERY  (Ecosia)
- maps QUERY    (Google Maps)
- ytm QUERY     (YouTube Music)
- yt QUERY      (YouTube)
- wikipedia QUERY
- yahoo QUERY
